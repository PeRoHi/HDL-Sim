# HDL-Sim UI かんたん起動

プログラミングに慣れていない人向けの起動手順です。

## 1. 最初に一度だけ準備

Python 3.12 を入れたあと、ターミナル/コマンドプロンプトで次を実行します。

```bash
python3 -m pip install fastapi uvicorn lark pytest
```

Windows では `python3` の代わりに `python` または `py` を使ってください。

## 2. 起動

### macOS / Linux

`start-ui.command` をダブルクリック、または:

```bash
./start-ui.sh
```

### Windows

`start-ui.bat` をダブルクリックしてください。

### どの OS でも使える方法

```bash
python3 start_ui.py
```

起動するとブラウザが自動で開きます。開かない場合は、表示された URL（通常 `http://127.0.0.1:8765`）をブラウザに貼り付けてください。

## 3. 使い方

1. 画面中央のエディタに Verilog を書く
2. 左上の **▶ 実行** を押す（または F5）
3. 左で階層、右で波形、下で `$display` 出力を確認する
4. サンプルは上部の **例** から読み込めます

## 終了方法

起動した黒い画面/ターミナルで `Ctrl+C` を押してください。

## うまく起動しないとき

- `fastapi` や `uvicorn` が無いと言われたら、準備コマンドをもう一度実行してください。
- ポート `8765` が使用中でも、自動で近い空きポートを探します。
- ブラウザが開かない場合は、ターミナルに表示された URL を手動で開いてください。
